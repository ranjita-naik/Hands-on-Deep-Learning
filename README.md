## 🧠 Overview

This repo is a curated collection of **end-to-end deep learning pipelines** implemented in Google Colab, covering:

- Supervised Learning (Tabular, Vision, Audio)
- Convolutional Neural Networks (CNNs)
- Transfer Learning
- Natural Language Processing (Embeddings, Transformers)
- Large Language Models (LLMs)
- Retrieval-Augmented Generation (RAG)
- Fine-tuning (LoRA)
- Generative Models (Diffusion Models)

Each notebook is designed to be:
- Self-contained
- Reproducible
- Beginner-friendly but production-aware
